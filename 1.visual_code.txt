COMANDOS EDITOR VISUAL CODE

Abrir pelo prompt de comando = code .

(Vários cursores no texto) = ctrl + shift + seta  ou alt + click

(Paleta de cores) = control + shift + L 

(Duplicar linhas) = shift + alt + up or down 

(Mover linhas) = alt + up or down 

(Deleta uma linha inteira) = ctrl + shift + k 

(Língua Portuguesa) = ctrl + shift + p > display

(Instalar visualização instantânia no browser) = extensão Live Server.
